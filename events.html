💡 Create a Custom Calendar

Want to create your own calendar with selected events? Use the Event Type values shown in the table:

  1. Find events you want to include in your calendar
  2. Copy the Event Type (partition_key) - click the copy icon to easily copy it
  3. Edit configs/calendars.json and add a new entry with your event types
  4. Run python -m sharksice to generate your custom calendar

See the home page or README for detailed examples.
